# 📋 Task Manager (Trello Clone)

Fullstack веб-приложение для управления досками и задачами. Учебный проект.

## 🛠️ Технологии

**Backend:** Node.js, Express, SQLite, JWT  
**Frontend:** Vue 3, Vue Router, Pinia, Axios, Bootstrap 5

## ✨ Функционал

- Регистрация и авторизация (JWT)
- Создание, просмотр, удаление досок
- Создание, удаление задач
- Адаптивный дизайн на Bootstrap
- Выход из аккаунта

## 🚀 Запуск проекта локально

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Открой `http://localhost:5173`
